# gulu-xingkongs

## 介绍
这是我在学习 Vue 过程中做的一个 UI 框架

## 开始使用

1.使用本框架前，请在本css 中开启 border-box

     *{
        box-sizing:border-box;
     }


2.你需要声明默认变量

    html {
      --button-height: 32px;
      --font-size: 14px;
      --button-bg: white;
      --button-active-bg: #eee;
      --border-radius: 4px;
      --color: #333;
      --border-color: #999;
      --border-color-hover: #666;
    }


3.引入 index.css
